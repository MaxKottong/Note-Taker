# Note Taker

## Purpose
When given a prebuilt website, I want to be able to connect clients to a
server and vice-versa so that they are able to take notes and save/delete 
them on the app.

## Built with
* Short-unique-id
    * `npm i short-unique-id`
* express.js
    * `npm i express`
* [Heroku](https://devcenter.heroku.com/)
